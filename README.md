# Navbar

SaaS для онлайн-записи клиентов к барберам и мастерам красоты. Рынок — Узбекистан.

Полное описание продукта: [docs/project-brief.md](docs/project-brief.md)
Архитектурные правила (мульти-рынок, i18n, валюты): [docs/architecture.md](docs/architecture.md)

## Структура репозитория

```
web/      — веб-страница бронирования (Next.js), navbar.uz/[slug]
mobile/   — мобильное приложение (Flutter, Android + iOS)
docs/     — бриф и документация
supabase/ — миграции БД и Edge Functions, появятся позже
```

## Как запустить локально

Нужен установленный [Node.js](https://nodejs.org) версии 18 или новее (проверить: `node --version`).

```bash
# 1. Склонировать репозиторий (один раз)
git clone https://github.com/javadabdullaev97-coder/navbar.git
cd navbar/web

# 2. Установить зависимости (один раз)
npm install

# 3. Запустить
npm run dev
```

Открыть в браузере: <http://localhost:3000>

Демо-страница мастера: <http://localhost:3000/asror>

## Как поставить приложение на Android

APK собирается автоматически при каждом изменении в `mobile/`:

1. Открой [Releases](https://github.com/javadabdullaev97-coder/navbar/releases) → релиз **apk-latest**
2. Скачай `app-release.apk` на телефон
3. Открой файл — Android спросит разрешение на установку из неизвестных источников, разреши
