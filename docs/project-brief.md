# Navbar — Project Brief для Claude Code

## Что это

SaaS-приложение для онлайн-записи клиентов к барберам и мастерам красоты. Рынок — Узбекистан (старт в Ташкенте), потом Казахстан и Кыргызстан. Основной конкурент — DIKIDI, но без локализации под UZ. Ниша свободна.

---

## Название и брендинг

**Navbar** (рабочее название). Логотип в прототипе: `nav` + `bar` (вторая часть акцентным цветом #A8FF78). Можно менять.

---

## Стек

| Слой | Технология | Причина |
|---|---|---|
| Мобильное приложение | Flutter | iOS + Android из одной кодовой базы, 70% UZ на Android |
| Бэкенд / БД | Supabase | Auth + PostgreSQL + Realtime + Storage из коробки |
| Веб-страница бронирования | Next.js | Публичная страница мастера, клиент записывается без скачивания приложения |
| Уведомления | Telegram Bot API | Бесплатно, клиент получает напоминания в Telegram |
| Оплата подписки | Payme + Click + Uzum | Локальные узбекские платёжки, карты UZ |
| Хостинг | Vercel + Supabase Cloud | Бесплатно на старте |

## Роли в приложении

Одно приложение, две роли. При первом входе пользователь выбирает роль — и видит разный интерфейс.

### Роль: Мастер (платит подписку)
- Ведёт расписание
- Управляет базой клиентов
- Смотрит аналитику доходов
- Получает новые записи от клиентов
- Настраивает услуги, цены, график работы
- Получает публичную страницу по ссылке (web booking page)

### Роль: Клиент (бесплатно)
- Видит своих мастеров
- Записывается через приложение или по веб-ссылке
- Получает напоминания в Telegram
- Видит историю визитов

## Экраны — Мастер

### Онбординг
1. Выбор роли (Мастер / Клиент) — большие карточки
2. Ввод номера телефона +998 → OTP через Telegram или SMS
3. Заполнение профиля: имя, фото, специализация, адрес, slug для ссылки

### Главный экран — Расписание
- Приветствие с именем мастера
- Мини-статистика: кол-во записей сегодня, выручка сегодня, кол-во клиентов
- Горизонтальный скролл с датами недели (активная дата выделена)
- Список записей на выбранный день: время, длительность, имя клиента, услуга, цена, статус (подтверждена / ожидает)
- FAB-кнопка добавить запись вручную

### Клиенты
- Счётчик лимита на бесплатном тарифе (38/50)
- Поиск по имени / номеру
- Список клиентов: имя, телефон, дата последнего визита, количество визитов
- Карточка клиента: история визитов, заметки мастера, предпочтения

### Аналитика (только платный тариф)
- Переключатель периода: неделя / месяц / 3 месяца
- Крупная цифра выручки за период
- Столбчатый график по дням
- Разбивка по услугам: название, кол-во раз, сумма

### Настройки
- Блок текущего тарифа + кнопка апгрейда
- Профиль мастера
- Моя страница (slug, ссылка для копирования)
- Услуги и цены
- График работы
- Настройка уведомлений клиентам

---

## Экраны — Клиент

### Главный экран
- Ближайшая запись (большая карточка): мастер, услуга, время, дата, адрес
- Список "Мои мастера" с кнопкой "Записаться" у каждого
- История прошлых визитов

### Запись к мастеру
- Выбор услуги
- Выбор даты и времени из доступных слотов
- Подтверждение
- Привязка Telegram для напоминаний

## Web Booking Page (Next.js)

URL вида `navbar.uz/[slug]` — например `navbar.uz/asror`

Что отображается:
- Фото мастера, имя, специализация, адрес
- Список услуг с ценами и длительностью
- Календарь доступных слотов
- Форма записи: имя клиента + номер телефона
- После записи: подтверждение + предложение скачать приложение

Клиент не скачивает ничего — просто открывает ссылку в браузере.

## База данных (Supabase / PostgreSQL)

```sql
-- Пользователи (мастера и клиенты)
users (
  id uuid PK,
  phone text UNIQUE,          -- +998XXXXXXXXX
  role text,                   -- 'master' | 'client'
  name text,
  avatar_url text,
  telegram_id bigint,          -- для уведомлений
  lang text DEFAULT 'ru',      -- 'ru' | 'uz' | 'en'
  created_at timestamptz
)

-- Профили мастеров
masters (
  id uuid PK REFERENCES users,
  slug text UNIQUE,            -- navbar.uz/[slug]
  bio text,
  address text,
  lat float, lng float,        -- геолокация
  subscription_tier text DEFAULT 'free', -- 'free' | 'pro' | 'salon'
  subscription_until timestamptz,
  booking_enabled bool DEFAULT true,
  clients_count int DEFAULT 0  -- кэш для лимита
)

-- Услуги мастера
services (
  id uuid PK,
  master_id uuid REFERENCES masters,
  name text,
  duration_min int,
  price int,                   -- в сумах
  is_active bool DEFAULT true,
  sort_order int
)

-- График работы
schedules (
  id uuid PK,
  master_id uuid REFERENCES masters,
  day_of_week int,             -- 0=пн, 6=вс
  start_time time,
  end_time time,
  is_day_off bool DEFAULT false
)

-- Клиентская база мастера
clients (
  id uuid PK,
  master_id uuid REFERENCES masters,
  user_id uuid REFERENCES users NULLABLE, -- если клиент установил приложение
  name text,
  phone text,
  notes text,
  visit_count int DEFAULT 0,
  last_visit_at timestamptz,
  created_at timestamptz
)

-- Записи
appointments (
  id uuid PK,
  master_id uuid REFERENCES masters,
  client_id uuid REFERENCES clients,
  service_id uuid REFERENCES services,
  starts_at timestamptz,
  ends_at timestamptz,
  status text DEFAULT 'pending', -- 'pending' | 'confirmed' | 'done' | 'cancelled'
  notes text,
  source text DEFAULT 'manual',  -- 'manual' | 'web' | 'app'
  created_at timestamptz
)

-- Уведомления (лог)
notifications (
  id uuid PK,
  appointment_id uuid REFERENCES appointments,
  type text,                   -- 'reminder_2h' | 'confirmation' | 'cancellation'
  sent_at timestamptz,
  channel text                 -- 'telegram' | 'sms'
)
```

## Тарифная система

### Бесплатный
- База клиентов: до 50 человек
- Расписание и запись: да
- Публичная страница (web booking): да
- Уведомления клиентам: нет
- Аналитика: нет
- Предоплата: нет

### Про — 29 900 сум/мес (~$2.3)
- База клиентов: безлимит
- Всё из бесплатного
- Автоуведомления клиентам в Telegram (за 2 часа до записи)
- Аналитика доходов
- Рассылки по базе
- Настройка предоплаты (ссылка напрямую на Payme/Click мастера)

### Салон — от 59 900 сум/мес
- 2–3 мастера: 59 900 сум
- 4–5 мастеров: 89 900 сум
- 6–10 мастеров: 129 900 сум
- Всё из Про
- Единый журнал салона
- Учёт выручки по мастерам
- Расчёт зарплат / процентов
- Роли: владелец / мастер / администратор

Оплата подписки: Payme, Click, Uzum. Мастер НЕ принимает оплату от клиентов через платформу — клиенты платят напрямую мастеру. Платформа берёт только подписку.

## Уведомления — логика

1. Мастер настраивает: уведомлять за X часов до записи (по умолчанию 2 часа)
2. Клиент при записи вводит номер телефона → ему предлагается подключить Telegram бот
3. За 2 часа до записи Supabase Edge Function отправляет сообщение через Telegram Bot API
4. Если нет Telegram — SMS через local SMS gateway (Eskiz.uz или Playmobile)
5. Мастер тоже получает уведомление о новой записи

## MVP — порядок разработки

**Этап 1 (2 недели)**
- Flutter проект, навигация, смена роли
- Supabase: auth по телефону + OTP, таблицы users / masters / clients / services / appointments / schedules
- Экран расписания мастера (просмотр)
- Добавление записи вручную

**Этап 2 (2 недели)**
- Web booking page (Next.js): профиль мастера + форма записи
- Клиентская база с лимитом (50 на free)
- Экран клиента: мои мастера + ближайшая запись

**Этап 3 (2 недели)**
- Telegram Bot для уведомлений
- Supabase Edge Function для отправки напоминаний
- Интеграция Payme для оплаты подписки
- Проверка тарифа (subscription_until) через Row Level Security

**Этап 4 (месяц 2)**
- Аналитика мастера
- Рассылки по базе клиентов
- Тариф Салон (мультимастерность)
- Полировка UI, i18n (узбекский)

## Конкурентный контекст

- **DIKIDI** — главный конкурент по СНГ, есть в UZ но без локализации и приёма оплаты в сумах. Бесплатный базовый тариф — наша основная конкуренция.
- **Masters (masters-app.ru)** — то что использует барбер заказчика. Без UZ-маркетинга.
- **YCLIENTS** — дорого, для больших салонов, в UZ почти нет.
- **Zapis.kz** — казахстанский аналог, только KZ.

Наши преимущества: локальные платёжки (Payme/Click/Uzum), узбекский язык, дешевле конкурентов в долларах (~$2.3/мес), поддержка в UZ часовом поясе.

## Ключевые бизнес-метрики для отслеживания

- MRR (Monthly Recurring Revenue) в сумах и USD
- Конверсия free → paid (цель: 25-30%)
- Churn в месяц (цель: <8%)
- CAC (стоимость привлечения мастера)
- DAU/MAU мастеров
